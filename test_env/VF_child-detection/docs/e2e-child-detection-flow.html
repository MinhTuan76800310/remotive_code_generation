B — Remotive CLI (thủ công) Observed

Pattern theo remotive_command.md và remotive-cli-cheatsheet.md. URL: http://localhost:50051 (nginx topology-api).

Setup một lần

export URL=http://localhost:50051

remotive broker restbus add --url $URL --frame SEAT-CpdCan0:SeatWeightSensor --start
remotive broker restbus add --url $URL --frame DMS-CpdCan0:CameraInput --start
remotive broker restbus add --url $URL --frame COCKPIT-CpdCan0:HmiDriverAction --start

Subscribe (terminal 2)

remotive broker signals subscribe --url $URL \
  --signal SEAT-CpdCan0:SeatInput.SeatOccupied \
  --signal DMS-CpdCan0:CameraInput.ChildDetectedByCamera \
  --signal DMS-CpdCan0:CameraInput.IsMoving \
  --signal CENTRAL-CpdCan0:ChildAlert.ChildAlertActive \
  --signal COCKPIT-CpdCan0:HmiChildWarning.ChildAlertActive \
  --signal AIRBAG-CpdCan0:AirbagActuatorState.AirbagActuated

Ví dụ TC-07 (alert ON)

remotive broker restbus update --url $URL --signal SEAT-CpdCan0:SeatWeightSensor.WeightKg:4.0
remotive broker restbus update --url $URL \
  --signal DMS-CpdCan0:CameraInput.ChildDetectedByCamera:1 \
  --signal DMS-CpdCan0:CameraInput.IsMoving:1
# Kỳ vọng: ChildAlertActive=1, HmiChildWarning.ChildAlertActive=1

Ví dụ TC-02 (alert OFF, sum=0.7)

remotive broker restbus update --url $URL --signal SEAT-CpdCan0:SeatWeightSensor.WeightKg:75.0
remotive broker restbus update --url $URL \
  --signal DMS-CpdCan0:CameraInput.ChildDetectedByCamera:1 \
  --signal DMS-CpdCan0:CameraInput.IsMoving:0

TC-09 — Airbag off (full chain)

# Sau khi alert ON (TC-07):
remotive broker signals publish --url $URL \
  --signal COCKPIT-CpdCan0:HmiDriverAction.TurnAirbagOff:1
# Kỳ vọng: AirbagActuatorState.AirbagActuated=1

Reset

remotive broker restbus stop  --url $URL --namespace SEAT-CpdCan0
remotive broker restbus stop  --url $URL --namespace DMS-CpdCan0
remotive broker restbus reset --url $URL --namespace SEAT-CpdCan0

Rủi ro & giới hạn

Observed centralhpc + topology mode: model khai báo namespace DMS-CpdCan0 trên cùng một BrokerClient trỏ tới broker CENTRAL/topology. Trên broker chỉ có CENTRAL-CpdCan0, container có thể Exited (1) với Namespace 'DMS-CpdCan0' is not configured.
Pytest qua topology-broker + restbus vẫn có thể pass nếu CENTRAL process chạy và bus UDP multicast đồng bộ namespace — cần xác nhận bằng ./run-e2e-tests.sh trên máy bạn.
Inferred Codegen bug đã sửa tạm trong test_env: centralhpc/__main__.py dùng self._*_latched trong WeightedLogOdds (trước đó thiếu self.). Nên port fix vào bmgen template để regen không ghi đè.
Unknown Single-broker mode (một broker, 6 namespace) — đã thử compose-single-broker.yml, broker Erlang crash multi-chain; chưa có workaround production trong repo.
